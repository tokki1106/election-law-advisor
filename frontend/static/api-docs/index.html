API 엔드포인트

POST /api/search 인증 불필요

선거 관련 법률 조문과 사례를 검색합니다.

{
  "query": "기부행위 제한",
  "top_k": 5,
  "expand_refs": true
}

curl 예시:

curl -X POST https://zugu.io/election/api/search \
  -H "Content-Type: application/json" \
  -d '{"query":"기부행위 제한","top_k":5,"expand_refs":true}'

응답 예시:

{
  "query": "기부행위 제한",
  "total": 7,
  "results": [
    {
      "id": "공직선거법_법률__제112조",

        "law_name": "공직선거법_법률",

AI 에이전트 연동

Claude / ChatGPT / Gemini

아래 프롬프트를 AI에게 전달하면 됩니다:

선거법 관련 질문에 답할 때 https://zugu.io/election/api/search 를
POST 호출하여 관련 법 조문을 검색한 뒤 근거로 활용해줘.
요청 형식: {"query": "검색어", "top_k": 5, "expand_refs": true}
개인정보 보호: 이 API는 IP 주소, 검색 쿼리를 로깅하지 않습니다. 인증 없이 익명으로 사용 가능합니다. LLM 호출이 포함되지 않으므로 비용이 발생하지 않습니다.
